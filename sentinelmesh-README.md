## What this is
A fully static, single-page demo dashboard showing all 6 SentinelMesh modules with live-animated mock data. No backend, no dependencies, no build step required.

## How to deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `sentinelmesh`)
2. Upload all files in this folder to the repo root
3. Go to **Settings > Pages**
4. Set source to **Deploy from a branch**, branch: `main`, folder: `/ (root)`
5. Click **Save** — your site is live at `https://YOUR-USERNAME.github.io/sentinelmesh` within ~60 seconds
